## 📖 Project Overview

This project focuses on building an **intelligent image captioning system** using the Flickr8k dataset.  
It combines **Computer Vision + Natural Language Processing (Multimodal AI)** to automatically generate meaningful captions from images.

The project implements:

- ✅ CNN + LSTM baseline model  
- ✅ Transformer-based BLIP model (fine-tuning)  
- ✅ Beam Search caption generation  
- ✅ Evaluation with BLEU, METEOR, CIDEr, SPICE  
- ✅ Explainable AI (Grad-CAM)

---

## 📊 Dataset

- 📂 Dataset: Flickr8k  
- 🖼️ 8,091 images  
- 📝 5 captions per image  

---

## 🏗️ Project Structure

In general, a multimodal AI project for image captioning using the Flickr8k dataset, combining CNN-LSTM baseline and BLIP fine-tuning with evaluation metrics and explainable AI techniques.
